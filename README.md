# U.S. -Data-Analyst-Job-Market-Dashboard
## Overview
Interactive Power BI dashboard analyzing 939 data analyst job postings 
pulled from the Adzuna API in March 2026.

## Key Findings
- Median salary: $102,500
- ETL and Data Modeling are the most in-demand skills
- Only 12% of roles are remote
- California and Texas lead in job volume

## Tools Used
Python · Pandas · Adzuna API · Power BI

## Files
- fetch_jobs.py — API data pull
- clean_jobs.py — data cleaning and transformation
- jobs_clean.csv — cleaned dataset (939 rows)
- skills_clean.csv — skill frequency table
- companies_clean.csv — top 30 hiring companies

## Key Findings
- ETL and Data Modeling are the top demanded skills which are more than SQL or Python alone
- Median salary for Data Analyst roles is $102,500, with 25% of roles paying $140k+
- Only 12% of roles are fully remote despite high Candidate demand for flexibility
- California accounts for the highest job volume, 
  followed by Texas and Washington
